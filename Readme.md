Software Maestro의 첫 번째 과제입니다.
==============
###이 과제는 프론트엔드+백엔드로 이루어져있으며, 
###프론트엔드는 유니티를 export해서 안드로이드에서 작동하고, 백엔드는 아파치톰캣 자바서블릿을 사용하였습니다.


####프론트엔드

1.유니티
- 버전 : 5.1.3f1
- 환경 : 윈도우10
- 위치 : Unity폴더 내

2.안드로이드
- 버전 : 1.5.1
- 환경 : 윈도우10
- 위치 : Android폴더 내


####백엔드

1.아파치톰캣자바서블릿
- 버전 :
 - 아파치 : 2.4.7
 - 톰캣 : 7
- 환경 : 우분투14.04 LTS (AWS)
- 위치 : temp폴더 내

2.mysql
 - 버전 : 14.14 Distrib 5.5.50
 - 환경 : 우분투14.04 LTS (AWS)
